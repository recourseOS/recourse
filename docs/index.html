SUMMARY
  Unrecoverable: 2 resources
  With effort:   1 resource

Recoverability Tiers

unrecoverable Data is gone. S3 without versioning, RDS without snapshots.
recoverable-from-backup Restore from snapshot or backup. RDS with snapshots, DynamoDB with PITR.
recoverable-with-effort Can recreate from config/code. Lambda, IAM roles, EC2 from AMI.
reversible Just flip it back. Security group rule changes, instance stops.
