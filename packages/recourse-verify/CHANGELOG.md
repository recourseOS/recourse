# Changelog

## 0.1.0 - 2025-05-02

Initial release.

- Ed25519 signature verification for RecourseOS attestations
- RFC 8785 JSON Canonicalization Scheme (JCS)
- Key registry caching with rollback protection (§5.5)
- Batch verification with pre-fetch for consistency
- Trusted instances allow-list with URL normalization
- Cross-check verification between embedded and URL copies
- Zero runtime dependencies
